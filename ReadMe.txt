First Git Repositroty file.
Commmand:
git init: init a repository directory
git add: add file
git commit -m "comment": add version ID & comment
git reset --hard HEAD^ (option: HEAD^ back to previous version
				HEAD^^ second-previous version
			        HEAD~100 back to 100th version)
git reset --hard version-ID (go forward or backward)
git log: check log file & version id
git status
git diff readme.txt: check difference
git reflog: check previous version id if version deleted
